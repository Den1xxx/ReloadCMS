You have downloaded the beta version of ReloadCMS-1.5.0, the final release of which I plan in mid-July 2012.


I decided to call this release ReloadCMS 1.5.0, because due to the development of various numbers of years, 1.2.8, 1.2.9, 1.3.1, 1.4, were employed unfinished beta release, which probably never will be incomplete.
A call the next release of fc somehow confusing - not clear versioning product.
The owner reloadcms.com NightFly forgive, if you accidentally ruin the process.

What's new in 2012 compared to the version ReloadCMS-2011 (describing the most important)
1. utf-8 instead of the set cp- for  languages
2. Improved caching. Let me remind you - the site caching works, but only for the guests.
- No cached pages a certain range - say guests are allowed to write in the forum, the forum does not mean the page cache.
- Made the cache reset button, as its volume can be quite significant.
3. The visual editor mode "html" - TinyMCE
4. bb-codes switch to TinyMCE and back "on the fly."
5. Loading images in both editors without page reload.
6. Disabling statistics (for heavy sites).
7. Some scripts are copied to the jQuery, including script lightbox, which made him finally stable.

And even in small things are different features that enhance comfort and stability.
Suggestions are made to help the project - http://fromgomel.com/?module=sendmail&get=development
Bugs reported here - http://reloadcms.com/main/?module=forum&action=topic&id=132


Installation.
For Windows. Upload files to the server. Go to the home page and register the first user. He is the administrator.
For Unix. Upload files to the server. Set the permissions 777 on the root directory (non-recursively) and folders "backups", "config", "content", "uploads" (recursively). Go to the home page and register the first user. He is the administrator.
Learn more about settings see here: http://reloadcms.com/wiki/doku.php?id=reloadcms:en:start
Later (in the stable release) will be paved with an installer that will do everything for you.

Вы скачали бета-версию ReloadCMS-1.5.0, окончательный релиз которой я планирую в середине июля 2012г.


Я решил назвать этот релиз ReloadCMS 1.5.0, потому что из-за разработок различных лет номера 1.2.8, 1.2.9, 1.3.1, 1.4 оказались заняты недоделанными бета-релизами, которые вероятнее всего так и не будут доделаны.
А называть очередной релиз f.c. как-то путает — непонятна версионность продукта.
Владельца reloadcms.com NightFly прошу простить, если случайно испортил малину.

Что нового в 2012 по сравнению с версией ReloadCMS-2011 (описываю самое важное)
1. utf-8 вместо множества cp- для каждого языка
2. Доработано кеширование. Напомню - кеширование сайта работает, но только для Гостей.
- Не кешируются страницы определённого диапазона - допустим разрешено Гостям писать на форуме, значит страницы форума не кешировать.
- Сделана кнопка сброса кеша, т.к. его объём может быть довольно значительным.
3. Визуальный редактор в режиме "html" — TinyMCE 
4. bb-codes переключается в TinyMCE и обратно "на лету".
5. Загрузка картинок в обоих редакторах без перезагрузки страницы. 
6. Отключение статистики (для высоконагруженных сайтов).
7. Некоторые скрипты переписаны на jQuery, в том числе скрипт lightbox, что сделало его наконец-то стабильным в работе.

И по-мелочи еще разные фишки, повышающие удобство и стабильность.

Установка. 
Под Windows. Закачиваем файлы на сервер. Заходим на главную страницу сайта и регистрируем первого пользователя. Он и будет администратором.
Под Unix. Закачиваем файлы на сервер. Выставляем права 777 на корневую директорию(нерекурсивно) и папки backups, config, content, uploads (рекурсивно). Заходим на главную страницу сайта и регистрируем первого пользователя. Он и будет администратором.
Подробнее об установке чиитайте здесь: http://reloadcms.com/wiki/doku.php?id=reloadcms:ru:start
Позднее (в стабильный релиз) будет выложен установщик, который все сделает за Вас.
